# ImageTrade (Front)
Мой пет-проект на React. Фронт для для веб приложения, где пользователи могут создавать, продавать и покупать за виртуальную валюту алгоритмически сгенерированные картинки.

## Стек:
- JS
- React 18.2.0
- Bootstrap 4
